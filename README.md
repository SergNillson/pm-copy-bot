# 🤖 Polymarket Copy Trading Bot

Автоматический бот для копирования сделок успешных трейдеров на Polymarket с режимом paper trading.

## 🎯 Что это?

Этот бот отслеживает сделки выбранного трейдера на платформе Polymarket и автоматически копирует их на ваш аккаунт с пропорциональным масштабированием позиций.

**Пример:** Если у трейдера баланс $2,000,000 и он делает ставку $10,000 (0.5% баланса), а у вас $40, бот сделает ставку $0.20 (тоже 0.5% от вашего баланса).

## ✨ Основные возможности

- ✅ **Paper Trading режим** - тестирование без риска
- ✅ **Пропорциональное масштабирование** - сохранение процентного соотношения
- ✅ **Отслеживание P&L** - виртуальная прибыль/убыток
- ✅ **История сделок** - все сделки в SQLite базе
- ✅ **Подробное логирование** - полная прозрачность работы
- ✅ **Простая настройка** - один YAML файл

## 🚀 Быстрый старт

### Шаг 1: Клонирование репозитория
```bash
git clone https://github.com/SergNillson/pm-copy-bot.git
cd pm-copy-bot
```

### Шаг 2: Создание виртуального окружения
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate
```

### Шаг 3: Установка зависимостей
```bash
pip install -r requirements.txt
```

### Шаг 4: Настройка конфигурации
Скопируйте файл примера и отредактируйте под себя:
```bash
cp config/.env.example config/.env
```

Отредактируйте `config/config.yaml` (по умолчанию уже настроен на тестовый режим)

### Шаг 5: Запуск бота
```bash
python main.py
```

## ⚙️ Конфигурация

Основные настройки в `config/config.yaml`:

```yaml
trader:
  target_wallet: "0x63ce342161250d705dc0b16df89036c8e5f9ba9a"  # Кошелек для отслеживания

paper_trading:
  enabled: true           # Paper trading режим
  virtual_balance: 40.0   # Начальный виртуальный баланс

trading:
  min_bet_usd: 0.01      # Минимальная ставка
  max_bet_usd: 10.0      # Максимальная ставка
  check_interval: 60     # Интервал проверки (секунды)
```

## 🔍 Как это работает

1. **Мониторинг**: Бот периодически проверяет новые сделки целевого трейдера через Polymarket API
2. **Расчет**: Вычисляет пропорциональный размер позиции на основе вашего баланса
3. **Симуляция**: В paper mode виртуально "исполняет" сделку и записывает в БД
4. **Отчетность**: Показывает текущий P&L и статистику

## 📝 Просмотр результатов

Результаты paper trading сохраняются в `data/paper_trades.db`. 

Для просмотра базы данных можно использовать:
- [DB Browser for SQLite](https://sqlitebrowser.org/) (GUI)
- Командная строка: `sqlite3 data/paper_trades.db`

**Это версия с paper trading!** Никакие реальные сделки не совершаются. Для реальной торговли потребуется:
- API ключи Polymarket
- Подключение кошелька
- Дополнительная настройка (будет в следующих версиях)

## 🛠 Troubleshooting

**Бот не запускается:**
- Проверьте, что установлен Python 3.8+: `python --version`
- Убедитесь, что активировано виртуальное окружение
- Переустановите зависимости: `pip install -r requirements.txt --upgrade`

**Нет новых сделок:**
- Проверьте адрес кошелька в `config/config.yaml`
- Убедитесь, что трейдер активен
- Проверьте логи в `logs/bot.log`
